# Archived Scripts Documentation

## Description

All Python scripts in this folder were originally located in the `opencompass/examples/` directory.

## Archive Date

**Feb 17, 2025**

## Reason for Archiving

These scripts are not related to the main functionality of the current project. They were moved to this archive folder to clean up the working directory.

## Important Notes

- **All script code remains unmodified**, only the file locations were changed
- These scripts can still be used normally. To use them, please move them back to the `examples/` directory or adjust the import paths
- The only files retained in the `examples/` directory are:
  - `eval_emoa.py`
  - `eval_emoa_agg.py`
