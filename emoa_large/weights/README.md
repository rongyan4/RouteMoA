# Router Weights

The router weight file for the large model pool (`router_large.pth`) is not included in this repository due to file size constraints.

## Download

Download the pre-trained router weights from Google Drive:

**[router_large.pth — Google Drive](YOUR_GOOGLE_DRIVE_LINK_HERE)**

After downloading, place the file in this directory:

```
emoa_large/weights/router_large.pth
```

This file is required only for the **RouteMoA** method. MoA and SMoA do not need router weights.
